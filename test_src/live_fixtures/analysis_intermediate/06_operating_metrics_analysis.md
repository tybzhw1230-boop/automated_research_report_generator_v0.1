# 中微半導體（深圳）股份有限公司 运营指标分析中间产物

## 1. 核心 operating metrics 与口径边界

基于 File Source、Search Source 及业务/行业分析，本次分析采用的核心运营指标体系如下：

*   **核心指标清单**：
    *   **规模与效率类**：总出货量（Volume）、平均售价（ASP）、细分产品销量（MCU/SoC/ASIC）。
    *   **客户与生态类**：客户留存率、服务客户总数、分销商数量。
    *   **研发与壁垒类**：研发人员占比、自主 IP 核数量。
    *   **市场地位类**：中国 MCU 市场出货量占有率。
*   **口径说明与限制**：
    *   **ASP 差异**：File Source 披露的 2025 年 ASP（0.28 元）与 Search Source 测算的 2023 年 ASP（0.61 元）存在显著差异，主要系 File Source 包含了 2025 年最新的产品组合降价及大客户策略性定价影响。
    *   **销量单位**：公司披露单位为“千件”，同行对比多采用“亿颗”，换算关系为 1,000 千件 = 0.01 亿颗。
    *   **行业补充指标**：晶圆采购价格波动率、库存周转天数（DOI）及 Design-win 数量为行业关键指标，但公司目前仅部分披露或未披露，存在数据缺口。

## 2. 公司 operating metrics 趋势分析

*   **出货量爆发式增长，呈现“以价换量”特征**：
    *   公司总出货量从 2023 年的 18.1 亿颗增长至 2025 年的 40.6 亿颗，年复合增长率极高。
    *   **原因**：2024-2025 年行业进入复苏期，公司通过策略性降价（ASP 从 0.39 元降至 0.28 元）快速抢占家电及消费电子市场份额。
*   **产品结构向高价值量 SoC 偏移**：
    *   SoC 销量从 2023 年的 0.7 亿颗增长至 2025 年的 2.3 亿颗，增速远超通用 ASIC。
    *   **原因**：符合业务分析中提到的“芯片+算法”一站式方案转型，SoC 集成度高，虽 ASP 也有所下滑（1.31 元降至 1.05 元），但仍显著高于通用 MCU。
*   **客户粘性与生态壁垒持续增强**：
    *   客户留存率从 63.5% 稳步提升至 82.8%。
    *   **原因**：公司拥有超 1,000 个自主 IP，通过底层算法绑定，提高了客户的转换成本。分销商数量从 78 家增至 88 家，渠道渗透力加强。
*   **研发投入保持高位**：
    *   研发人员占比达 52.5%，支撑了车规级 MCU（BAT32A 系列）等高门槛产品的快速迭代。

## 4. operating metrics 与业务/行业逻辑的一致性分析

*   **与行业周期一致**：2023 年 ASP 较高但销量低（去库存压力），2024-2025 年销量激增但价格下行，符合行业分析中提到的“2023 低谷，2024 复苏”及国产替代初期的价格战逻辑。
*   **与业务模式一致**：公司 Fabless 模式下，2025 年 1-2 月出货量已达 8.26 亿颗，反映了其与中芯国际等供应商签定的产能保障协议有效支撑了高频交付。
*   **与增长战略一致**：车规级产品收入增速达 88.7%，虽然目前在总营收占比仅 4.8%，但其高 ASP 特征是未来扭转公司整体 ASP 下行趋势的关键。

## 5. 关键缺口与未确认信息

*   **毛利率与成本缺口**：File Source 提到 2023-2025 年单位存货成本降低 27.9%，但未披露具体的晶圆采购单价波动。考虑到对供应商 A（中芯国际）54.3% 的采购集中度，**上游议价权缺失**是否会抵消技术降本的效果需进一步确认。
*   **ASP 统计冲突**：Search Source 测算的 2023 年 ASP（0.61 元）与 File Source 披露的 2023 年 ASP（0.39 元）存在较大偏差，可能源于是否包含功率器件或其他非 MCU 产品的统计口径差异。
*   **海外收入真实性**：File Source 称 2025 年收入 100% 来自国内，而 Search Source 显示有 4.8% 境外收入。这涉及公司全球化布局（香港/新加坡）的实际进展，对评估地缘政治风险至关重要。
*   **库存水位**：未披露具体的库存周转天数（DOI），在行业强周期背景下，无法准确判断当前 40 亿颗的出货量中，有多少是终端消耗，有多少是渠道压货。
