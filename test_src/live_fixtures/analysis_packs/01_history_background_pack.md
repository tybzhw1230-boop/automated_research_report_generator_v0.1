# 中微半導體（深圳）股份有限公司 历史与背景分析

## 1. 公司基本信息与定位
- **1.1 公司全称、注册地、成立日期、上市/拟上市状态**
    - **公司全称**：中微半導體（深圳）股份有限公司（英文：CHINA MICRO SEMICON (SHENZHEN) LIMITED / Cmsemicon (Shenzhen) Corp., Ltd.）。
    - **注册地/总部地址**：存在两个主要记录地址，分别为：1. 深圳市前海深港合作区南山街道桂湾三路91号景兴海上大厦2102室；2. 深圳市南山区粤海街道高新区社区科苑南路3099号中国储能大厦3101。
    - **成立日期**：2001年6月22日（另有记录为2001年3月27日）。
    - **上市状态**：已于2022年8月5日在上海证券交易所科创板上市（证券代码：688380.SH）；目前正申请在香港联交所主板上市。
- **1.2 主营业务定位与核心产品/服务概述**
    - 公司是一家领先的集成电路设计企业（Fabless模式），专注于数模混合信号芯片及模拟芯片的研发与销售。以MCU（微控制器）为核心，提供包括芯片、算法及技术支持在内的一站式智能控制解决方案。
- **1.3 行业归属与市场定位**
    - **行业归属**：半导体集成电路设计。
    - **市场定位**：专注于消费电子、智能家电、工业控制及汽车电子等领域，致力于高效能、具能源效益及成本效益的国产替代方案。

## 2. 发展时间线与关键里程碑
- **2.1 创立与早期发展**
    - 2001年：公司前身在深圳成立，由杨勇先生创立，开启MCU研发业务。
    - 2017年：成立成都研发中心，扩张研发版图。
- **2.2 重大业务转型、并购、融资事件**
    - 2018年：收购四川芯联发，启动自有组装与测试生产线，实现设计与封测协同。同年引入顺为科技、南海成长等机构投资者。
    - 2021年：技术突破，实现12英吋晶圆90/55nm eFlash MCU量产。
    - 2023-2024年：加大汽车电子投入，获ISO26262汽车功能安全认证，切入车规级供应链。
- **2.3 上市/拟上市历程**
    - 2019年12月：整体变更为股份有限公司（另有记录显示股改完成于2020年12月）。
    - 2022年8月5日：A股科创板上市。
    - 2026年（拟）：提交H股上市申请。
