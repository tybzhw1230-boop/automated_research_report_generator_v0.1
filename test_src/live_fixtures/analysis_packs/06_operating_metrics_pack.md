# 中微半導體（深圳）股份有限公司 运营指标分析

## 1. 核心 operating metrics 定义与口径
本次分析基于公司披露及行业惯例，采用以下核心运营指标体系：
- **规模类指标**：总出货量（包含MCU、SoC、ASIC等芯片产品合计销售数量）、细分产品销量。
- **价格类指标**：平均售价（ASP，定义为营业收入除以同期相应销量）。
- **客户与渠道类指标**：客户留存率（往绩期间持续交易客户占比）、服务客户总数、授权分销商数量。
- **效率与技术类指标**：研发人员占比（研发人员数/总员工数）、自主IP核数量。
- **市场地位类指标**：中国MCU市场出货量占有率。

## 5. 公司 operating metrics 趋势分析
- **“以价换量”特征明显**：总出货量从2023年的18.1亿颗爆发式增长至2025年的40.6亿颗，同期ASP从0.39元降至0.28元。这表明公司在行业复苏期通过策略性降价快速抢占市场。
- **产品结构向高价值量偏移**：SoC销量增速远超通用ASIC，从0.7亿颗增至2.3亿颗。尽管SoC单价也有所下滑，但1.05元的ASP仍远高于通用MCU，符合公司“芯片+算法”的转型战略。
- **生态壁垒持续增强**：客户留存率从63.5%提升至82.8%，结合超1,000个自主IP，显示公司通过底层算法绑定提高了客户转换成本。

## 6. 同行业对比下的相对位置与原因
- **相对位置**：中微半导是典型的“量大价低”型选手。在出货量上具备规模优势（2024年中国MCU出货量占比12.6%，排名第一），但在收益维度和单品价值量上与兆易创新等龙头仍有差距。
- **原因分析**：
    1. **应用领域差异**：中微深耕家电和消费电子，此类市场容量大但价格竞争激烈；而兆易创新在存储和高性能MCU领域有更高溢价。
    2. **成本结构**：公司通过缩减晶粒尺寸等技术手段降本，支撑了低价扩张策略。
    3. **供应链依赖**：对中芯国际的高度依赖（54.3%）虽然保障了产能，但也限制了在上游涨价时的议价空间。

## 7. 关键缺口与待确认事项
- **ASP 统计口径冲突**：Search Source 测算的 2023 年 ASP（0.61 元）与 File Source 披露值（0.39 元）存在偏差，需确认是否包含功率器件等非核心产品。
- **库存风险未知**：未披露库存周转天数（DOI），无法判断 40 亿颗出货量中是否存在渠道压货风险。
- **海外收入数据矛盾**：File Source 称 2025 年收入 100% 来自国内，与 Search Source 提到的 4.8% 境外收入存在冲突，需核实全球化布局真实进展。
- **成本传导机制**：单位存货成本虽降低 27.9%，但具体晶圆采购单价未公开，难以评估毛利率改善的持续性。
