# 中微半導體（深圳）股份有限公司 同行信息分析

## 1. 可比公司筛选标准与样本清单

### 1.1 筛选维度（主营构成、地域、规模、上市状态）
- **主营业务一致性**：入选公司均采用 **Fabless** 模式，核心产品均包含 **MCU（微控制器）** 或 **SoC（系统级芯片）**。
- **下游应用重合度**：优先选择在 **智能家电、电机控制、消费电子** 以及正在发力的 **车规级芯片** 领域有深厚布局的企业。
- **技术路径相似性**：中微半导强调“控制+模拟+功率”及“芯片+算法”策略，因此选取了在电机驱动算法（峰岹科技）和数模混合设计（芯海科技）方面具有代表性的同行。
- **上市状态**：所有样本均为 A 股科创板或主板上市公司，财务透明度高，具备可比性。
- **排除逻辑**：显式排除了 IDM 厂商（如华润微）、纯模拟/功率厂商（如圣邦股份）以及国际巨头（如 ST、Renesas），以确保财务对标的有效性。
