# 中微半導體（深圳）股份有限公司 可比估值分析包

## 1. 相对估值结论摘要
- **当前采用的核心倍数**：**P/S (EV/Revenue)** 与 **P/B**。
- **为什么这些倍数有代表性**：
    1. **P/S (EV/Revenue)**：中微半导目前处于“以价换量”的扩张期，2023-2024年间因行业周期性波动和研发高投入出现利润大幅波动甚至亏损，PE倍数受非经常性因素干扰较大。P/S能更客观反映其在家电及电机控制领域的市场份额及营收规模。
    2. **P/B**：公司资产结构极度稳健，货币资金及理财占总资产比例近80%，且有息负债为0。在盈利不稳定的阶段，PB倍数提供了坚实的资产价值底座，反映了市场对其充足现金储备的估值认可。

## 3. 调整逻辑
- **溢价因素**：
    - **市场地位**：中微半导2024年中国MCU出货量占比12.6%，排名第一，具备显著的规模效应和客户粘性（留存率82.8%）。
    - **资产安全性**：拥有约21亿现金储备，抗风险能力远高于芯海科技（负债率55%）等同行。
    - **技术协同**：其“芯片+算法”策略在电机控制SoC领域具有较强护城河，可比性直指峰岹科技。
- **折价因素**：
    - **盈利韧性**：ASP（平均售价）下滑较快（0.39元->0.28元），毛利率受行业去库存冲击比兆易创新更敏感。
    - **产品结构**：目前仍高度依赖家电及消费电子，车规级芯片营收占比（<5%）虽在增长，但尚未形成业绩支柱，相比国芯科技在汽车领域的先发优势略显不足。

## 4. 可比性局限
- **盈利状态不一**：样本中芯海科技、国芯科技处于亏损状态，导致PE/EBITDA中位数可能产生偏离。
- **业务重心差异**：兆易创新侧重存储+高性能MCU，乐鑫科技侧重Wi-Fi SoC，而中微半导更偏向数模混合与家电控制，应用场景的差异决定了毛利率上限的不同。
- **数据滞后性**：由于2026E预测数据缺失，当前的估值主要基于历史TTM数据，未完全计入车规级产品放量后的业绩弹性。这要求投资者在解读时，需动态关注ASP止跌信号及车规Design-win的兑现情况。
