# 中微半導體（深圳）股份有限公司 内在价值分析包

## 1. 估值框架与核心假设
- **估值方法**：采用两阶段折现现金流（DCF）模型。鉴于公司目前处于“以价换量”的转型期且拥有极高的净现金储备，DCF 能更好地捕捉其研发投入转化为未来现金流的潜力，而非受短期亏损干扰。
- **关键假设**：
    1. **预测期增长率**：基于 2025 年出货量爆发式增长（40.6 亿颗）但 ASP 下滑的背景，Base 情景假设未来 5 年收入复合增长率为 15%，反映家电 MCU 稳固地位及车规级芯片的放量。
    2. **自由现金流（FCF）基数**：以 2023A EBITDA（约 1,012 万元）为参考，考虑研发费用资本化潜力及未来盈利修复，设定初始 FCF。
    3. **折现率 (WACC)**：设定为 11%，反映半导体行业高波动性及地缘政治风险溢价。
    4. **终值增长率**：设定为 2%，符合成熟半导体设计环节的长期增长预期。
    5. **净现金**：公司拥有约 21 亿元货币资金及理财，这是估值的重要支撑。
- **假设强度说明**：
    *   **强证据**：净现金储备（财务报表明确）、出货量增长趋势（经营指标明确）。
    *   **弱证据**：长期毛利率修复路径（受制于晶圆成本和 ASP 竞争）、车规级芯片的实际利润贡献（目前占比仅 4.8%）。

## 4. 关键结论与置信度说明
- **内在价值区间**：**22.80 亿元 - 36.12 亿元**。考虑到公司目前市值与净资产、净现金的关系，该区间显示出较强的下行保护（Safety Margin），主要由 21 亿现金支撑。
- **对哪些变量最敏感**：**利润率（毛利率）的修复**。目前公司处于亏损边缘，FCF 的微小变动会对估值产生巨大杠杆效应。其次是**折现率**，地缘政治对供应链（中芯国际依赖度 >50%）的影响直接决定了风险溢价。
- **哪些关键假设缺乏支撑**：
    1. **ASP 止跌时点**：2025 年 ASP 仍在显著下降（0.38->0.28），尚无证据表明价格战已结束。
    2. **车规级产品盈利性**：虽然销量增长，但未披露具体毛利，无法确认其是否能拉动整体利润率。
- **为什么需要收窄或下调置信度**：由于 **2024A 详细利润数据缺失** 且 **2025 年存在“以价换量”的极端经营特征**，导致 FCF 基数的预测存在较大偏差。在看到扣非净利润转正信号前，应维持“中低”置信度，估值更偏向于资产价值（现金）而非营运价值。
