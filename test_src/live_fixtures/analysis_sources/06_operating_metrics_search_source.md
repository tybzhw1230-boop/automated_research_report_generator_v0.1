  - **缺失的关键期间或口径**：
    1. **细分产品单价**：各公司年报通常披露总销量，但很少拆分 MCU、SoC、模拟芯片的各自单价，导致不同产品结构的 ASP 对比存在“苹果比橘子”的偏差（如乐鑫科技的高单价 SoC 与中微半导的低单价通用 MCU 无法直接等同）。
    2. **产能/封测数据**：由于均为 Fabless 模式，不披露产能利用率，且委外加工费用的明细口径（如测试费与封装费的拆分）在各家财报中披露不一。
    3. **2024Q3 实时销量**：季度报告通常只披露财务数据，不披露具体的销量和单价，导致最新期间的运营指标存在滞后。

  - **原因说明**：虽然细分产品口径有差异，但在“总销量”和“平均单价”这两个核心维度上，上述公司均有连续两年的公开数据，足以反映出 2022-2023 年半导体周期下行对国产 MCU 行业的共同影响（即“量增价跌”的去库存特征）。研发人员数量的对比也能有效衡量各公司的技术投入强度。
