# 中微半導體（深圳）股份有限公司 风险分析包 File Source

## ENTRY F_BUS_045
- 问题：扩产项目风险
- 期望输出：扩产项目的建设风险及产能消化风险。
- 输出内容：公司目前正计划搬迁遂宁生产线，预计于2026年末前完成新厂房的装修工程及搬迁。虽然遂宁生产线主要负责新开发产品在工程批次的包装与验证测试，对集团收益贡献较低（往绩期间低于5%），但搬迁过程中的装修进度、设备调试及产能平移仍存在不确定性。

## ENTRY F_RISK_001
- 问题：行业下行风险
- 期望输出：需求下滑、周期见顶、增长放缓。
- 输出内容：IC行业具有周期性波动，历来会出现扩张与收缩模式，伴随产品需求减少、产能过剩、高存货水平及售价下跌。2023年公司财务表现受到行业整体衰退、下游需求疲软的不利影响，录得相对较低的毛利及毛利率。

## ENTRY F_RISK_002
- 问题：市场竞争加剧风险
- 期望输出：新进入者、替代品、价格战、份额流失。
- 输出内容：全球IC市场竞争激烈，现有市场参与者扩大规模及新竞争者加入将使竞争进一步加剧。竞争对手可能拥有更雄厚的财务资源、更广的客户群及更高的品牌知名度，能更快推出新产品或施加价格压力，导致公司盈利能力下降及市场份额流失。

## ENTRY F_RISK_003
- 问题：技术替代风险
- 期望输出：技术路径变化导致产品被替代。
- 输出内容：IC行业技术进步迅速，产品迭代频繁。技术标准演变可能导致公司现有技术、解决方案或产品竞争力下降或被淘汰。若技术或市场需求出现预料之外的转变，现有产品的商业相关性可能降低。

## ENTRY F_RISK_004
- 问题：需求结构变化风险
- 期望输出：下游需求结构变化的风险。
- 输出内容：下游行业（消费电子、智能家电、工业控制及汽车电子）受消费者行为转变、法规变动、贸易限制及供应链重组等结构性因素影响。例如消费电子仍处于去存货阶段，而汽车电子进入补存货阶段，需求趋势的分化对公司产品组合提出挑战。

## ENTRY F_RISK_005
- 问题：市场空间不及预期
- 期望输出：行业增速低于预期的风险。
- 输出内容：公司业务依赖下游行业的长期增长，若消费电子、智能家电等领域发展放缓，或技术进步（如人工智能整合、低功耗技术）市场接受度延迟，将限制行业增长空间，对公司收益和长期前景造成不利影响。

## ENTRY F_RISK_006

## ENTRY F_RISK_007

## ENTRY F_RISK_008

## ENTRY F_RISK_009
- 问题：原材料涨价风险
- 期望输出：原材料大幅上涨且无法转嫁。
- 输出内容：晶圆是公司最主要的原材料成本，2023-2025年占销售成本的59.8%、64.2%及59.5%。晶圆价格受全球需求、地缘政治等因素影响，美元升值也会增加采购成本。若无法将成本转嫁给客户，将压缩利润空间。

## ENTRY F_RISK_010
