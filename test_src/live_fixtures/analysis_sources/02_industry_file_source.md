## ENTRY F_IND_002
- 问题：细分领域
- 期望输出：各个大类行业的主要细分市场及各细分市场的特征。
- 输出内容：
    - **MCU（微控制器）**：
        - **特征**：集成CPU、内存及标准接口的“一体化”芯片；结构简单，低功耗、低成本、低复杂度；执行基本通用控制任务（如温度传感、电机控制）。
        - **细分**：8位MCU（针对低成本、低功耗基本控制）和32位MCU（提供更高处理速度、大内存，应对复杂算法）。
    - **SoC（系统级芯片）**：
        - **特征**：高度集成，将多个功能模块（CPU、GPU、通讯、AI加速等）整合于单芯片；支持多任务和并行处理，性能敏感。
    - **ASIC（专用集成电路）**：
        - **特征**：完全定制化电路，专为单一用途设计；效率和性能最高，但无法修改应用范围；针对大容量、高性能应用。
    - **主要应用细分市场**：
        - **消费电子**：对成本敏感，需中等运算性能。
        - **智能家电**：需触控、电机控制等功能，强调可靠性。
        - **工业控制**：需高可靠性、耐用性及安全连接。
        - **汽车电子**：准入门槛最高，需符合AEC-Q100等严苛安全与质量标准。

        - **下游关系**：公司通过分销和直销模式，将智能控制解决方案提供给终端设备制造商（如家电品牌、汽车制造厂）。
